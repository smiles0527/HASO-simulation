# Documentation Index

This directory contains detailed documentation for the HASO Emergency Evacuation Simulator.

## Getting Started

**GETTING_STARTED.md** provides a quick introduction to running simulations, understanding key concepts, and customizing the simulator. Includes code examples for common tasks and API quick reference.

## Operational Guides

**HOW_TO_RUN.md** offers step-by-step instructions for executing simulations using various methods: scenario launcher, quick tests, visual demos, Jupyter notebooks, and custom scripts.

## Algorithm Documentation

**HASO_ALGORITHM.md** describes the Hierarchical Adaptive Search and Optimization algorithm, including mathematical formulations for zone partitioning, agent assignment, and dynamic adaptation.

**ELECTRICAL_FLOW_MODEL.md** explains the circuit-theory-based flow dynamics model, with equations for pressure initialization, resistance calculation, flow rates, and route optimization.

**OBSTACLE_SYSTEM.md** details the obstacle and hazard system, covering edge types, dynamic state management, hazard propagation models, and integration with pathfinding.

## Summary

**IMPROVEMENTS_SUMMARY.md** provides an overview of key features, performance characteristics, and research applications of the simulator.
